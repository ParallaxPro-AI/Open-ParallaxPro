# Game Template Index

One-line summary of every template in `reference/game_templates/`. Use this
to pick the closest starting point by name, then `Read` that template's 4
JSONs (`01_flow.json`, `02_entities.json`, `03_worlds.json`,
`04_systems.json`) to copy patterns from.

The two templates pre-read during warming are **rts_battle** (single-player
exemplar) and **buccaneer_bay** (multiplayer exemplar).
